## 🚀 Functionality

### Gallery Layout

- Displays images in a clean grid layout.
- Automatically adjusts according to screen size.

### Lightbox

- Click any image to open it in a fullscreen lightbox.
- Close the lightbox using the close button or outside click.

### Navigation

- Previous button
- Next button
- Smooth image switching

### Responsive Design

- Desktop
- Laptop
- Tablet
- Mobile

### Hover Effects

- Image zoom animation
- Smooth transition effects
- Interactive user experience

### Bonus Features

- Filter images by category
- Smooth filtering animations
